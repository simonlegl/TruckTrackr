> Find food trucks. Anytime. Anywhere.

TruckTrackr is a real-time food discovery platform designed for university students and food truck vendors. It bridges the gap between hungry students and mobile kitchens with live GPS tracking, integrated ordering, and a robust loyalty system.

## Features

### For Students
*   **Live Map**: Real-time GPS tracking of food trucks on campus.
*   **Online Ordering**: Browse menus and order ahead to skip the line.
*   **Loyalty Rewards**: Earn points with every purchase and redeem for free food.
*   **AI Assistant**: 24/7 chatbot support for menu queries and order status.

### For Vendors
*   **Analytics Dashboard**: Track sales, revenue, and peak hours.
*   **Visibility**: Dedicated profile with menu management and location sharing.
*   **Marketing**: tiered subscription plans for increased visibility and insights.

## Tech Stack

*   **Frontend**: React 19, Tailwind CSS
*   **Icons**: Lucide React
*   **Charts**: Recharts
*   **AI**: Google Gemini API (via `@google/genai`)

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Create a `.env` file with your Google GenAI API Key:
    ```
    API_KEY=your_api_key_here
    ```
4.  Run the development server:
    ```bash
    npm start
    ```
